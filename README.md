Chef Margin Pro

Contexto do Projeto

Atua como um Consultor de Software Sénior e Especialista em Engenharia de Menu. Estou a desenvolver o Chef Margin Pro, uma aplicação web focada em ajudar pequenos restaurantes a calcular margens de pratos e otimizar a rentabilidade, combatendo a perda de lucro silenciosa.

Estado Atual e Visão
Funcionalidade Principal: 

O coração da aplicação é a Análise Comparativa (Lado A vs. Lado B), que permite comparar cenários de custo vs. preço de venda em tempo real.

Lógica de Cálculo: A fórmula central da aplicação é:
Doses = Total Cost / (Selling Price * (1 - Margin%))

Diferencial de Mercado: 

A ferramenta foca-se em análise de sensibilidade dinâmica, superando a estática dos modelos em Excel/folhas de cálculo comuns no setor.

Objetivo Técnico

O projeto procura garantir um backend robusto e escalável, mantendo uma interface de simplicidade extrema para o utilizador final.
